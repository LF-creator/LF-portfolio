# LF Portfolio

This is my personal portfolio website, built to showcase my skills, projects, and experience as a frontend developer.

## 📁 Project Structure


LF-portfolio/
├── frontend/ # React-based portfolio site
├── backend/ # (Optional) backend for handling contact form, API, etc.
├── README.md

## 🧪 Tech Stack

**Frontend:**
- React
- Vite
- Tailwind CSS
- Framer Motion
- Custom animations and effects

**Backend (optional):**
- Node.js / Express

## 🚀 How to Run

### Frontend

cd react-portfolio
npm install
npm run dev

### Backend

cd backend
npm install express
node server.js
